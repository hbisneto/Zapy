# Zapy

Envio de mensagens através do WhatsApp


## Recomendações

* Python 3.x ou superior;
* É recomendada a execução do programa em ambiente virtual;

### VirtualEnv:

```
pip install virtualenv
```

### pywhatkit

```
pip install pywhatkit
```

* Python 3.x ou superior
